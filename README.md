
# CapitolOwl

A comprehensive financial analysis and investment education platform designed to help new investors grow their confidence through interactive data visualization and community-driven learning.




## Tech Stack

**Frontend:** 
- Next.js – React framework for server-side rendering and API routes
- Tailwind CSS – Utility-first CSS for rapid UI development

**Backend:** 
- FastAPI – High-performance Python framework for RESTful APIs
- TBD – (e.g., MongoDB, PostgreSQL, or Supabase depending on requirements)

**Development Tools:** 
- Axios – HTTP client for API requests
- Docker – Containerization for consistent development and deployment

## Features
- Interactive Data Visualization – Explore stock trends, financial metrics, and performance through dynamic charts and graphs.

- Investment Education Modules – Learn fundamental and technical analysis with informative cards.

- Community blog – Share and discuss your investment stories, strategies, and analysis with other users.



## Installation

**Install and run the frontend**

```bash
cd frontend
npm i 
npm run dev
```

**Install and run the backend**
```
cd backend
python -m venv ./venv
pip install -r requirements.txt
uvicorn --reload main:app
```
